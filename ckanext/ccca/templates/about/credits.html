    Open source data platform

    The CCCA Data Server is built with CKAN, an open-source data-publishing platform.
    The CCCA Data Centre has extended and customized CKAN according to its needs.
    The code is licensed under the terms of the GNU Affero, and the extensions are going to be published on github.
